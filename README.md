# sistema-validacao-senha-java
Sistema simples de validação de senha em Java.
Este é um projeto simples desenvolvido em Java que simula a validação de uma senha.

## O que foi utilizado:
- Estrutura de repetição (while)
- Entrada de dados com Scanner
- Validação de senha

## Como executar:
Execute o programa e insira a senha até acertar o valor correto.
